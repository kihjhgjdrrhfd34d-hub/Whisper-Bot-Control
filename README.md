# Whisper Bot Control

A powerful Telegram Whisper Bot with advanced management features.

## Features

- Anonymous whisper messages
- Reply system
- Admin panel
- Dashboard
- REST API
- Enterprise features
- Statistics
- Rate limiting
- Plugin system
- Automated tests

## Requirements

- Python 3.13+
- python-telegram-bot
- SQLite

## Installation

```bash
git clone git@github.com:kihjhgjdrrhfd34d-hub/Whisper-Bot-Control.git
cd Whisper-Bot-Control
pip install -r requirements.txt
```

## Configuration

Create a `.env` file:

```
BOT_TOKEN=your_token
ADMIN_ID=your_admin_id
```

## Run

```bash
python main.py
```

## Tests

```bash
pytest
```
